# Bomber 3D

3D‑игра в стилистике **"Bomber"**.  
Игрок двигается по клеточному полю, ставит бомбы, взрывает врагов и пробирается к выходу на противоположной стороне карты.

---

## Управление

**В игре (сцена Main):**

- `WASD` — движение по сетке.
- `Space` — поставить бомбу в текущей клетке.
- `Esc` — пауза / возврат из паузы.

**В меню:**

- **Панель настроек**:
  - Ползунки:
    - Player Speed (клеток/с) — скорость игрока  
    - Enemy Speed (клеток/с) — скорость врагов  
    - Bomb Radius (клетки) — радиус взрыва бомбы  
    - Enemy Spawn Interval (с) — частота спавна врагов
